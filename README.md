# pdbpipe.vim
experimental debugging python vim plugin

### Mapping

```
 <leader>7:  Startup debug.
 <leader>9:  toggle Breakpoint current line.
 <leader>5:  Continue.
 <leader>3:  Step.
 <leader>4:   Next.
```
